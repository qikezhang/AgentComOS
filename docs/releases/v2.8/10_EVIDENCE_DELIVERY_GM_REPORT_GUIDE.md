# 10 Evidence / Delivery / GM Report Guide

## Evidence Packet

For v2.8 final delivery, Evidence must include:

- Discord inbound message
- GM command
- permission check result
- executor policy
- executor command
- executor result
- adapter result
- audit log
- delivery packet
- GM report

## Delivery Packet

Delivery must never report blocked or rejected commands as completed.

## GM Report

GM Report must disclose:

- Docker service status
- Discord adapter status
- permission result
- executor policy result
- command status
- blocked reason
- audit summary
- next action
- incident or recovery actions if relevant
