# R2 Codex Review Checklist

- [ ] Branch is R2 branch.
- [ ] Dockerfile exists.
- [ ] docker-compose.yml exists.
- [ ] .env.example exists.
- [ ] No real .env committed.
- [ ] Healthcheck command exists.
- [ ] Runtime/logs/reports/backups volumes configured.
- [ ] Restart policy configured.
- [ ] make compile passed.
- [ ] make test passed.
- [ ] make validate-examples passed.
- [ ] No runtime artifacts committed.
- [ ] No secrets committed.
- [ ] Acceptance report updated.
