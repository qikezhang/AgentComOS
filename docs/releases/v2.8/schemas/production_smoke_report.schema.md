# Production Smoke Report Schema

## Required sections

- branch
- commit
- Docker Compose status
- healthcheck result
- Discord connection result
- Discord permission checks
- Controlled Executor checks
- Operation Adapter checks
- incident response checks
- backup/restore checks
- Evidence / Delivery / GM Report checks
- secrets scan
- runtime artifact cleanup
- final decision

## Final decision values

- passed
- failed
- partial
