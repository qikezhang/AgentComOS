# R8 Final Release Spec

## Goal

Promote release candidate to v2.8.0 final.

## Required conditions

- rc1 smoke passed.
- no blocking issues.
- Codex final release review passed.
- documentation complete.
- rollback instructions complete.
- incident response complete.
- backup/restore complete.

## Required actions

- create final release notes
- create tag `v2.8.0`
- publish final delivery package
- archive acceptance reports

## Out of scope

- post-v2.8 features
- Worker Evolution
- Auto Versioner
